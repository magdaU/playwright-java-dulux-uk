🎭 Playwright UI Automation Framework (Java)

This is a UI test automation framework built with Java, Playwright, JUnit 5, and Maven.
It is designed for reliable end-to-end testing of modern web applications.

🧰 Tech Stack
Java 21
Playwright
JUnit 5
Maven
AssertJ
🏗 Project Structure

The framework is organized using the Page Object Model (POM) with additional support for reusable components.

🚀 Purpose

This framework is used for:

End-to-end UI testing
Regression testing
Cross-browser validation
Automating critical user flows

The focus is on stability, readability, and maintainability.
